Task manager

Task management is the tool for creating and Managing tasks through the project’s lifecycle.
It includes gathering requirements, planning, status tracking, testing and creating final reports when tasks are completed.
Individuals use task management tools like a pen and a pad or software tools to organize and accomplish personal goals for every day’s chores.
Tasks could have status, start date, due date, people who are assigned to work on them, comments, tags and files which are attached to them.
I have used AngularJS, NodeJS, MongoDB as for backend.
